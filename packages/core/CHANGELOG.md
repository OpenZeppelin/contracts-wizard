## 0.4.0 (2023-10-05)

### Breaking changes
- Update to OpenZeppelin Contracts 5.0. ([#284](https://github.com/OpenZeppelin/contracts-wizard/pull/284))

- Use token-specific pausable extensions.
- Enable ERC20Permit by default.
